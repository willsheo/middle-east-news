# Korean Terminology Glossary (용어집)

**Middle East Daily Briefing**

Pinned Korean renderings for recurring terms, so the Korean edition uses identical vocabulary every day. Consult this file when writing the Korean edition; when a new recurring term appears, add it here the same day (and regenerate the PDF). Do not improvise a different rendering for a term that already has a row.

Register: analytical written style, 한다체 (plain reportive). Numbers in Arabic numerals with Korean units where natural (약 8,470만 배럴, 10억 달러). Outlet names stay in their original form ([Al Jazeera](...), [Bloomberg](...)); links unchanged. Confidence ratings: **신뢰도: 높음 / 중간 / 낮음**.
